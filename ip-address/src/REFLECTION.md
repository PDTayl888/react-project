While porting this app to React I tried to take as many cues from the recent sba as possible. Kept it straightforward with only as many pages and components as necessary. Avoided over engineering meant I Kept the header plain jsx with only the searchbar as a component, mainly because it would neatly encapsulate the functionality as well and not clutter the header too much. I only had 2 custom hooks, one for fetch and one for localstorage, with corresponding contexts so everyone gets to use them. I was inspired by the react module labs and sba to add a favorites page. As far as styling I tried to think of it the same as last time just with some custom html elements this time. Debugging was a nightmare.
